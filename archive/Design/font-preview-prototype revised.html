Instance preview footer prototype

Preview and Naming order share one footer slot — never stacked, never both open. Naming order is the default on load. Switch to Preview to glyph-test the selected (or hovered) instance. No fvar sliders; location comes from the Instances panel.
  • Mode toggle in the footer header swaps the body content exclusively.
  • Naming tools (Code / Hide pinned / Restore) show only in Naming order mode.
  • Collapsed footer shows a one-line summary for whichever mode is active.
