Font Preview prototype

Separate window for visually testing the variable font — axis sliders + named-instance picker — so you can catch rendering / identity problems before export (the kind that bite undo in host apps when PS names or instance coordinates shift).
  • Source — open anytime from the on-disk font + live project axes. No vfcommit.
  • Planned — needs a temp materialize (same dry-run write path as Review, without opening Review). Core Text only sees new names/instances from real font bytes.
  • fvar axis ranges are never touched — the font is already compiled and this tool doesn't re-interpolate. Only STAT (stops), name, and the fvar instance list get rewritten.
  • New STAT stops can optionally be promoted to a selectable fvar named instance, or left STAT-only (visible to style-linking/menus, but not pickable here or via the fvar API). Both show up in Instance changes below.
  • Sliders drive live font-variation-settings. Instance dropdown snaps to named coords. Roboto Flex stands in for the project VF.
